# Pi2Sports Backend

Backend API for Pi2Sports built with Node.js, Express, and Stripe.

## Run locally
```bash
npm install
npm start
```

## Deploy on Render
- Set environment variables:
  - STRIPE_SECRET_KEY
  - STRIPE_WEBHOOK_SECRET
  - FRONTEND_URL (your Netlify site, e.g. https://pi2sports.netlify.app)
```
